    Introduction

    Welcome to my journey of developing "Cuisineous," a restaurant management app. Over the past weeks, I have been diligently working on various aspects of this project, from understanding Dart and Flutter to integrating Firebase for authentication and database management. This blog post will walk you through the weekly progress and learnings that shaped the development of Cuisineous.


    Week 1: Diving into Dart and Flutter

    Date: 15/05/2024 - 22/05/2024

    This week marked the beginning of my journey into mobile app development with Dart and Flutter. I focused on understanding the basics of Dart, including code flow, basic concepts, loops, and classes. Additionally, I explored Flutter and its fundamental widgets and layouts. By the end of the week, I was able to design a basic UI with common UI components and build a simple application.

    Key Learnings:

    • Dart language basics

    • Introduction to Flutter

    • UI design with Flutter widgets

    Next Week's Plan:

    • Work on project design

    • Develop Sign-in and Sign-out pages with validation

    • Create an Introduction Page


    Week 2: Project Design and Initial Development

    Date: 22/05/2024 - 29/05/2024

    The second week was focused on designing the overall structure of the project and starting the initial development of key pages. I developed the Sign-in and Sign-out pages, ensuring they included proper validation. Additionally, I created an Introduction Page to guide users through the app's features.

    Key Learnings:

    • Project structure design

    • Implementing user validation

    • Introduction page creation

    Next Week's Plan:

    • Create profile and edit profile pages

    • Design a drawer for the home page

    • Develop the menu page


    Week 3: Building the Foundation

    Date: 29/05/2024 - 05/06/2024

    In the third week, I concentrated on creating the profile and edit profile pages, designing a drawer for the home page, and making the menu page. A significant part of this week was dedicated to learning about Firebase, setting it up, and connecting the app to Firebase. I implemented database integration for the signup and login pages, storing data in Firestore.

    Key Learnings:

    • Firebase setup and integration

    • Profile and menu page creation

    • Implementing Firestore for data storage

    Next Week's Plan:

    • Connect more pages to the database

    • Develop additional menu pages

    • Create side drawer pages


    Week 4: Enhancing Functionality

    Date: 05/06/2024 - 12/06/2024

    The focus of this week was on expanding the functionality of the app. I added more menu pages and authenticated the app through Firebase. A settings page was created, accessible from the drawer, and Google Authentication was added for logging in. Additionally, I implemented a password reset feature.

    Key Learnings:

    • Firebase authentication

    • Google Authentication integration

    • Implementing password reset functionality

    Next Week's Plan:

    • Connect more menu pages to Firebase


    Week 5: Finalizing the App

    Date: 12/06/2024 - 19/06/2024

    In the final week of the internship, I focused on connecting more menu pages to Firebase and ensuring that all functionalities worked seamlessly. I also made final adjustments to the app's design and performance. This week marked the culmination of all the hard work put into developing the Cuisineous app.

    Key Learnings:

    • Advanced Firestore integration

    • Finalizing app functionalities

    • Performance optimization

    • Completing the app's design

    Internship Conclusion: The development of Cuisineous has been a challenging yet rewarding journey. Each week brought new learnings and experiences that contributed to the overall progress of the project. Working with Dart, Flutter, and Firebase, especially Firestore, has significantly enhanced my skills in mobile app development. This internship has provided me with practical knowledge and experience that will be invaluable in my future career.


    This blog post now includes a detailed account of the progress made during weeks 1 to 5, covering the key learnings and plans for each subsequent week, as well as a conclusion summarizing the entire internship experience.
